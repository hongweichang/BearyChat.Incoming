#BearyChat机器人Incoming SDK

[![Build status](https://ci.appveyor.com/api/projects/status/wyr12letout7kojh?svg=true)](https://ci.appveyor.com/project/vip56/bearychat-incoming)
[![NuGet](https://img.shields.io/nuget/v/Nuget.Core.svg?style=plastic)](https://www.nuget.org/packages/BearyChat.Incoming)

为了便于以后运维监控的即时通知（非邮件），将采用BearyChat进行通知，故开发了此SDK

## 使用说明
直接通过实例化BearyChatClient对象并通过Send多种重载方式即可。
